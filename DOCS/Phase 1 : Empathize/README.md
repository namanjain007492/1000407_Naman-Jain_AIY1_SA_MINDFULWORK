# 🧠 Empathize Stage

## 📌 Overview
The Empathize stage focuses on understanding user needs, problems, and experiences related to workplace stress.

## 🔍 Research Method
- Conducted a Google Form survey
- Collected responses from 7 participants
- Analyzed stress patterns, emotions, and coping methods

## 📊 Key Findings
- Most users experience moderate to high stress (3–5 level)
- Common triggers include deadlines, workload, and meetings
- Users feel anxious, overwhelmed, and tired
- Current coping methods are not fully effective

## 👤 User Understanding
- Created Empathy Map to understand user thoughts and emotions
- Developed Personas to represent different user types

## 📁 Files Included
- Empathy_Map.png → Visual representation of user feelings and behavior
- Key_Research_Insights.png → Summary of research findings
- Personas.pdf → Detailed user profiles
