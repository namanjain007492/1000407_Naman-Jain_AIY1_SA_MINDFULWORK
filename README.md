# 🌿 Mindful Work App – Stress-Free Offices

## 📌 Overview

The *Mindful Work App* is a user-centered design project developed using the Design Thinking process. It aims to help individuals manage stress effectively during work through quick, personalized, and interactive digital solutions.

This project includes research, problem definition, ideation, prototyping, and testing, resulting in a functional web-based prototype.

---

## 🎯 Objective

* Reduce workplace stress
* Provide quick relaxation techniques
* Improve mental well-being
* Deliver a simple and engaging user experience

---

## 🧠 Design Thinking Process

### 1. Empathize

* Conducted user research using Google Forms
* Collected stress-related data from users
* Created:

  * Empathy Map
  * Personas
  * Research Insights

---

### 2. Define

* Identified key problems:

  * High stress due to workload and deadlines
  * Ineffective break methods
  * Lack of time for relaxation
* Developed:

  * Insight Clusters
  * POV (Point of View)
  * HMW (How Might We) questions

---

### 3. Ideate

* Generated solution ideas:

  * AI Stress Coach
  * Guided Meditation
  * Mood-based suggestions
  * Stress tracking dashboard
  * Smart break reminders

---

### 4. Prototype

* Designed a functional web app prototype including:

  * Home Dashboard
  * Mood Detection
  * Meditation Timer
  * Relaxation (Music & Sounds)
  * Stress Tracker
  * AI Stress Coach (chat-based support)

---

### 5. Test

* Conducted user testing with multiple users
* Collected feedback:

  * Positive: simple, calming, useful
  * Negative: needs more customization and features
* Improved design based on feedback
